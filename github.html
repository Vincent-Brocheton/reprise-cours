Reprise des cours et utilisation de Github

Github, à quoi ça sert?

Github est une plateforme déstinée à faire du versionning de code. On va pouvoir l'envoyer sur un dépôt distant afin d'avoir en ligne une version de son code. (on va pouvoir le lire, voir les modifications apportées, récupérer son code sur un autre environnement mais aussi le partager)

Créer son premier dépôt

  1. Créer son compte sur Github
  2. Télécharger Git
  3. Installer Git et selectionner Visual Studio Code comme éditeur par défaut
  4. Ouvrir le dossier avec VSC et taper la commande git init dans le terminal
  5. Saisir les commandes git config --global user.email "xxx" et git config --global user.name "xxx"
  6. Créer le repository sur Github
  7. Taper la commande git remote add origin "url du repository"
  8. Indexer les changements et faire un commit
  9. Effectuer un Push (Envoi des modifications sur Github)
  10. Se connecter à son compte Github depuis VSC

Faire des modfications sur un dépôt Github

  1. Créer une nouvelle branche pour le développement de fonctionnalités
  2. Implémenter la nouvelle fonctionnalité
  3. On index et on commit
  4. On Push
  5. On va sur Github et on merge
